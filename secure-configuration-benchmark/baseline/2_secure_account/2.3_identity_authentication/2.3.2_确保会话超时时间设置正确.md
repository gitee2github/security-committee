### 2.3.2 确保会话超时时间设置正确

**级别：** 要求

**适用版本：** 全部

**规则说明：** 

设置合理的会话超时时间可以降低因管理员人为原因而导致系统被攻击者攻击的风险。

openEuler默认会话超时时间300秒，可根据实际场景修改该值，但不得大于300秒。

**检查方法：**

```
# grep "^export TMOUT" /etc/profile
export TMOUT=300
```

**修复方法：**

- 修改/etc/profile文件TMOUT字段，根据业务场景修改为合理的值：

  ```
  # vim /etc/profile
  export TMOUT=<seconds>
  ```

- 使用source命令使之生效：

  ```
  # source /etc/profile    
  ```
